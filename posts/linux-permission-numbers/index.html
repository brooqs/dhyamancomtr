Linux Permission Numbers

  • 0 = Empty
  • 1 = –x -> Execute
  • 2 = -w- -> Write
  • 3 = -wx -> Write & Execute
  • 4 = r– -> Read
  • 5 = r-x -> Read & Excute
  • 6 = rw- -> Read & Write
  • 7 = rwx -> Read,Write,
